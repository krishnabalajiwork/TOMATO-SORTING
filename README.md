# Tomato Sorting System 🍅

This project is an AI-based tomato quality inspection system that detects tomatoes in an image and classifies them as **Good** or **Bad**. The system uses **YOLO for tomato detection** and **EfficientNet for quality classification**, and it is deployed as an interactive web application using **Streamlit**.

Users can upload an image or capture one using the camera, and the application will automatically detect tomatoes, classify their quality, and display the results with bounding boxes and labels.
